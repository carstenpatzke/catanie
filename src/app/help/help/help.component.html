How to get started The Getting Started Guide gives a brief description on how to get started using the data catalog.

Where is my data? Your data is stored and can be accessed by logging into {{ facility }} using sftp to download a file. Alternatively, individual datafiles can be downloaded by clicking on a datafiles tab for a given dataset, however this is not recommended for large datasets

Where is my data? Firstly, the ingestion process makes data on PSI central servers, known to the catalog. Secondly, archiving makes a copy of the data in the storage facility. Once archivng is complete, the data can be retrieved from the storage facility using the retrieval workflow.

How can I publish a dataset? Select datasets in the dataset table and add to Cart. Once in Cart, click Publish and follow instructions.

Ingest Manual The Ingest Manual provides detailed information on how to make your data available to the catalog as well as archiving and retrieval of datasets.

Where are my proposals? Once logged in, you can see your proposals at Proposals

Where are my samples? Once logged in, you can see your samples at Samples
